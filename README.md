**TEAM BIOS**
Push txt files titled lastname.txt to the team bios folder. Also add lastname.png with a bio photo of your choice. Format the txt like so:

Name: John Smith
Major: Biomedical Engineering
Year: Senior
Years in the club: 2022-2026
Position: Mechanical Team Member
A little about me: I have doggo!!
Photo name: lastname.png

Then add lastname.txt to the index.txt document in the team bios folder.


**Photo Gallary**
To put new photos in upload them the the images folder. Make sure to title them Photo#.png (Photo1.png) with the next avalible number. Make sure to update the photo-count.txt with the number of photos there are. So if you added Photo60.png, update the photo-count.txt to be 60.
